# 🎓 Student Record Management System

## 📖 Project Overview

The Student Record Management System is a Python-based application that allows users to manage student records using file handling and Object-Oriented Programming (OOP). The program stores student data permanently in a JSON file and provides a simple menu-driven interface for performing different operations.

This project was developed as part of a Python internship to practice Python fundamentals, file handling, JSON handling, exception handling, and Git & GitHub.

---

## ✨ Features

- ➕ Add a new student record
- 📋 View all student records
- 🔍 Search a student by Roll Number
- ✏️ Update student information
- ❌ Delete a student record
- 💾 Store records permanently in a JSON file
- 📊 Generate a summary report in a text file
- 📈 Sort students by marks
- 🏆 Display the top 3 highest-scoring students
- ✅ Validate duplicate roll numbers
- ✅ Prevent empty names
- ✅ Validate numeric roll numbers
- 🎨 Colored terminal interface using Colorama

## ▶️ How to Run the Project

1. Clone or download the repository.
2. Open the project folder in Visual Studio Code.
3. Install Colorama:

```bash
pip install colorama
```

4. Run the program:

```bash
python main.py
```

## 📄 Report Details

The generated report includes:

- Total number of students
- Number of students in each department
- Highest-scoring student
- Date and time of report generation

The report is automatically saved as **report.txt**.
